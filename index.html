CSS Snippet Cheat Sheet

Ever have trouble recalling the exact syntax for your favourite CSS code? Find snippets of some CSS code on this page! Select any snippet below and it'll automatically select all of the code for you to copy.

Flexbox Row

Use these three properties to create a Flexbox row layout

.row {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
}

Flexbox Column

Use this to create a Flexbox column layout

.column {
    display: flex;
    flex-direction: column;
}

CSS Grid Layout

Build a 12-column layout using CSS Grid

.grid {
    display: grid;
    width: 100%;
    grid-template-columns: repeat(12, lfr);
}

Linear Gradients

This will create a background linear gradiant from top to bottom

.linear-gradient-background {
    background-image: linear-gradient(
        rgba(232, 102, 236, 0.3) 0%, rgba(232, 102, 236, 0.6) 100%)
    );
}

Box Transition Glow

Use box transition and box shadow to glow on hover

.code-card .card-header {
    border-radius: 8px;
    transition: all 0.5s ease-in-out;
}

.code-card:hover, 
.code-card:hover .card-header {
    box-shadow: inset 0px 0px 8px rgba(232, 102, 236, 1), 0 0 15px rgba(232, 102, 236, 1)
}

CSS Variables

Use CSS variables to assign a value to a custom property which you can call in the stylesheet by using the CSS variable

:root {
    --main-green: #17a76b;
}
.nav {
    background: var(--main-green);
}
